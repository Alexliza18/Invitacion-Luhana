# Música de fondo

Coloca aquí el archivo de audio con el nombre `song.mp3` (o actualiza la ruta
en `src/config/event.ts` → `music.src` si usas otro nombre/formato).

No hay ningún archivo de audio incluido por defecto: el botón de música
seguirá funcionando (no rompe la app), simplemente no sonará nada hasta que
agregues el archivo real.

La música empieza a sonar automáticamente en cuanto el usuario presiona
"Abrir Invitación" (ese clic es la interacción que los navegadores exigen
para permitir audio con sonido). El botón flotante sigue disponible para
pausarla o reanudarla en cualquier momento.
